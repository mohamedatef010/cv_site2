Welcome to your Multilingual Site Manager!

This dashboard allows you to manage all aspects of your profile website in real-time. The site supports three languages: Arabic, English, and Russian. Every text field is split into three inputs so you can enter the translation for each language.

Instructions
  • Click Save Changes at the top right of the screen (or in the floating button) to persist updates to the website.
  • Image uploads are saved inside the img/ directory (profile photos), img/portfolio/ (project gallery), img/awards/ (certificates), or img/testimonials/ (client screenshots) automatically.
  • You can view the live site anytime by clicking View Website.
